总结

一、函数

1. 函数的基本使用

1.1【理解】函数的作用

  • 提高代码编写效率, 代码重用

1.2【重点】函数的定义和调用

  • 函数定义：

    """
def 函数名():
    函数内部的代码块
"""

def say_hello():
    print("hello 1")

  • 函数调用：

    # 函数调用格式：函数名字()

say_hello()  # 函数调用

1.3【理解】函数的执行过程

1.4【记忆】函数的文档注释

  • 函数体内第一行多行注释即为函数的文档注释，主要写函数的描述信息

2. 函数的参数

2.1【理解】函数参数的作用

  • 函数参数，可以传递数据给函数内部，增加函数的 通用性

2.2【重点】函数参数的使用

  • 带参数函数的定义格式：

    def 函数名字(形参1，形参2，……):
    函数体代码块

  • 带参数函数的调用格式：

    函数名字(实参1，实参2，……)

2.3【记忆】形参的作用域

  • 作用域：变量起作用的范围

  • 形参的作用域：只在定义函数的代码块 中

3. 函数的返回值

3.1【理解】函数返回值的作用

  • 开发中，有时会希望 一个函数执行结束后，告诉调用者一个结果，以便调用者针对具体的结果做后续的处理

3.3【重点】通过return给函数设置返回值

"""
函数定义格式：
def 函数名():
    return 结果

函数调用格式：
返回值变量 = 函数名()
"""
# 函数定义
def func():
    return '^_^'

# 函数调用
ret = func()

3.4【记忆】函数默认返回值

  • 函数内部没有任何return语句，默认返回None，表示没有任何数据

  • return不设置返回值，默认也返回None

3.5【重点】return中断函数

  • 函数一旦执行return，return下一句往后的代码不会执行

4. 【重点】四种函数的类型

  • 无参数，无返回值

  • 无参数，有返回值

  • 有参数，无返回值

  • 有参数，有返回值

5. 【理解】函数的嵌套调用

6. 局部变量和全局变量

6.1【记忆】局部变量和全局变量的区别

  • 定义方式不同

    • 局部变量是定义在函数内部的变量
    • 全局变量是定义在函数外部的变量

  • 作用域不同

    • 局部变量只在定义所在的函数内部有效
    • 全局变量在所有函数内都有效

6.2【重点】通过global声明修改全局变量

  • 函数内修改全局变量：先global声明全局变量，再修改

7. 函数参数详解

7.1【知道】位置参数和关键字参数区别

  • 位置参数：按形参的位置，从左往右，一一匹配传递参数

  • 关键字参数：通过 形参=值 方式为函数形参传值，无需和形参位置一一对应

# 函数定义
def func(num1, num2):
    pass

# 函数调用
func(1, 2)  # 位置参数
func(num2=3, num1=1)  # 关键字参数

7.2【重点】默认形参(缺省形参)

  • 形参设置默认值则为缺省参数，也叫默认参数

  • 调用函数时，如果没有传入默认参数对应的实参，则使用默认值

  • 默认参数必须在普通参数的后边
# 函数定义
def func(a, b=20, c=10):
    print(a, b, c)

# 函数调用
func(1)  # a=1,b=20,c=10
func(1, 2)  # a=1,b=2,c=10
func(1, 3, 2)  # a=1,b=3,c=2

7.3【重点】元组型不定长参数

  • 形参变量名前面加上一个*，这个参数则为元组型不定长参数
# *args 不定长参数，可以接收0~多个实参
# 把实参的1,2,3, 包装成元组(1, 2, 3)再传递, 等价于args = (1, 2, 3)
def func(*args):
    # 函数内部使用，无需加*
    print(args, type(args))


# 函数调用
func(1, 2, 3)  # (1, 2, 3) <class 'tuple'>

7.4【重点】字典型不定长参数

  • 定义参数时需要在形参名前添加**，则为字典型不定长参数
  • 字典型可变形参必须在形参列表的最后边
# 把实参包装成 {'city': 'sz', 'age': 18}给kwargs传递
# kwargs = {'city': 'sz', 'age': 18}
def func(name, **kwargs):
    # 存在形参name, name不会被包装到字典中
    print(name)
    print(kwargs)  # 函数内部使用，无需加*

# 实参的写法： 变量=数据，变量=数据
func(name='mike', city='sz', age=18)
