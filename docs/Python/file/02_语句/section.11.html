Python基础语言

总结

一、选择

1.【了解】选择应用场景介绍

  • 如果某些条件满足，才能做某件事情；条件不满足时，则不能做，这就是所谓的选择判断。

2.【理解】 比较和逻辑运算符

  1. 比较运算符

    • 2个数比较，满足结果为True, 不满足为False

      age = 15
print(age >= 18) # 结果为False

  2. 逻辑运算符

    • and：并且， 左右2边都为True，结果才为True
    • or：或者， 只有有1个为True，结果为True
    • not：非，取反，原来是True就变成False

3.【重点】if语句的基本使用

3.1 【重点】if语句的使用

"""
if 要判断的条件:
    条件满足时，要做的事情

步骤流程：
1. 定义一个变量age，保存年龄
2. if 判断条件(判断是否满 18 岁)：
    3. 满足条件，打印一句话：允许进网吧嗨皮
"""

age = 19
if age >= 18:
    print('允许进网吧嗨皮')

3.2【重点】if-else语句的使用

"""
if 要判断的条件:
    条件满足时，要做的事情
else:
    条件不满足时，要做的事情

步骤流程：
1. 定义年龄变量
2. if 判断是否满 18 岁:
    2.1 如果满 18 岁，打印一句话：允许进网吧嗨皮
3. 否则，未满 18 岁
    3.1 打印一句话：提示回家写作业
"""

age = 15
if age >= 18:
    print('允许进网吧嗨皮')
else:
    print('回家写作业')

3.3【记忆】通过if实现的三目运算符

  • 三目运算符：简洁版if-else语句
    • a if a > b else b : 如果 a > b的条件成立，三目运算的结果是a，否则就是b

3.4【重点】if-elif-else语句的使用

  • 条件从上往下判断

    • 只要满足一个条件，执行满足条件的代码块，其它条件不再判断执行
    """
语法格式：
if 条件1：
   满足条件1，执行的事情1
elif 条件2：
   满足条件2，执行的事情2
else:
   不满足上述条件，执行其它事情

步骤流程：
1. 定义 holiday_name 字符串变量记录节日名称
2. if 判断是否为情人节：
   2.1 如果是 情人节，打印： 买玫瑰／看电影
3. elif 判断是否为平安夜:
   3.1 如果是 平安夜，打印： 买苹果／吃大餐
4. else 其它情况：
   4.1 打印：每天都是节日啊……
"""

holiday_name = '情人节'
if holiday_name == '情人节':
   print('买玫瑰／看电影')
elif holiday_name == '平安夜':
   print('买苹果／吃大餐')
else:
   print('每天都是节日啊……')

4.【难点】if的嵌套

  • if 的嵌套：if里面还包含if
  • 写代码步骤：先写外层if，再写内层if

二、循环

1.【了解】循环应用场景介绍

  • 循环的作用就是让 指定的代码 重复的执行

2.【重点】while语句的基本使用

2.1【重点】while语句的使用

"""
1. 定义一个条件变量，一般赋值为0
2. while 判断条件：
    2.1 满足条件的代码块
    2.2 条件变量的改变【非常重要】
"""

i = 0
while i < 10:
    print('跑步第%d圈' % (i+1))
    i += 1

2.2【知道】死循环

  • 条件永远为真的(True)循环即为死循环，也叫无限循环

2.3【应用】计算1~100之间的累加和

"""
1. 定义条件变量 i, 赋值为1
2. 定义一个辅助变量，用于保存累计的结果
3. while 条件(i <= 100)：
    3.1 完成累加
    3.2 条件变量改变
4. 在循环的外面，打印最终的累加结果
"""

i = 1
_sum = 0

while i <= 100:
    _sum = _sum + i
    i += 1

print('_sum = %d' % _sum)

4. break 、 continue和循环的else

4.1【重点】break的作用

  • 跳出循环，不再执行循环体中后续重复的代码

4.2【重点】continue的作用

  • 跳过循环，不再执行本次循环体中后续重复的代码，但进入下一次循环判断

4.3【知道】循环的else执行流程

  • 循环里面没有遇到break语句，while执行完后，则会执行else的分支
