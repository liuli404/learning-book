Python基础语言

变量和类型

学习目标:

  1. 能够应用变量记录数据，例如：数量
  2. 知道至少 3 种 Python 常见数据类型，例如：整型、浮点型、字符串
  3. 能够使用 type 函数查看变量的类型

1. 变量

1.1 变量是什么

  • 变量源自数学，在计算机语言中变量可以理解为存储数据的容器
  • 程序是用来处理数据的，而变量是用来存储数据的

1.2 变量的定义

  • 在 Python 中，对变量 首次赋值会定义变量，再次赋值会修改变量的值

  • 等号=用来给变量赋值

    • = 左边是一个变量名
    • = 右边是存储在变量中的值
# 变量名 = 值

# 定义一个变量，名字叫num，赋值为123
num = 123
# 获取变量的值，打印显示在屏幕
print(num)

# 修改变量的值
num = 250
# 获取变量的值，打印显示在屏幕
print(num)

2.2 如何查看变量的类型

  • 在 Python 中定义变量是不需要指定类型，系统自动推导出变量中存储数据的类型
  • 可以使用 type(变量的名字) 来查看变量的类型
"""
250         int，整型
3.14        float 浮点型
'mike'      字符串，只要是''格式的内容就是字符串
"mike"      字符串，只要是""格式的内容就是字符串
True, False     bool, 布尔，计算True就是1， False就是0
"""

# 定义变量，无需指定类型，自动推导
a = 250
# type(变量名)  获取变量的类型，需要打印后，才能看到
print(type(a))  # 输出结果为：<class 'int'>

b = 3.14
print(type(b))  # 输出结果为：<class 'float'>

name = 'mike'
print(type(name))  # 输出结果为：<class 'str'>

is_man = False
print(type(is_man))  # 输出结果为：<class 'bool'>
