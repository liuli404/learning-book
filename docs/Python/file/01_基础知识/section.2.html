第一个 Python 程序

学习目标:

  1. 能够使用 PyCharm 完成第一个 Python 程序

1. PyCharm 介绍

PyCharm是 Python 的一款非常优秀的高效开发工具。

  • 支持多个操作系统
  • 适合开发大型项目

  • 官站：https://www.jetbrains.com/pycharm/

2. 第一次启动 PyCharm

2.1 导入配置信息

  • 如果是第一次使用，选择不导入以前配置，直接点击 OK 按钮

2.2 PyCharm 的配置主题界面

  • 在初始配置界面，可以通过 Editor colors and fonts 选择 编辑器的配色方案

  • 此外，还需要选择是否安装开发插件，直接跳过即可

3. 第一个Python程序

3.1 使用Pycharm编写程序

  • 打开 Pycharm，选择 Create New Project，创建一个新项目

  • 选择 Location 表示该项目保存的路径，Interpreter 表示使用的Python解释器版本，第一次需要手动配置解释器：

  • 右击项目，选择New，再选择Python File

  • 在弹出的对话框中输入的文件名hellowold，点击回车，表示创建一个Python程序的文本文件，文本文件后缀名默认.py

  • 输入以下代码，并右击空白处，选择Run运行，表示打印一个字符串'hello world'。

    print('hello world')

  • 运行成功后，Pycharm Console窗口将显示我们的输出结果。

3.2 练一练

要求：编写一个程序，输出itcast.cn

4. PyCharm工作区域和字体设置

4.1 PyCharm工作区域

  • 文件导航区： 浏览／定位／打开 项目文件
  • 代码编辑区： 编辑 当前打开的文件
  • 控制台区：
    • 输出程序执行内容
    • 跟踪调试代码的执行

5. PyCharm常用快捷键

  • 以下为windows平台下的快捷键
快键键 说明
Ctrl + D 复制选定的区域或行
Ctrl + Y 删除选定的行
